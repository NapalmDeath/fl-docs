# Как поделиться фотографиями, найденными на «Флатике»?

На «Флатике» легко поделиться понравившейся фотографией с друзьями или коллегами.  
Сделайте это в несколько шагов:

1. Наведите курсор на фотографию, которой хотите поделиться.  
2. Нажмите кнопку **Поделиться** в нижней части изображения.  
3. Выберите, где хотите поделиться фото — в соцсетях или по ссылке.  
4. Следуйте подсказкам во всплывающем окне.

> Скопированную ссылку можно отправить в мессенджере, по почте или сохранить для себя.
