# Как сделать альбом идей приватным?

Альбомы идей Флатики могут быть как публичными, так и частными. По умолчанию при создании они являются общедоступными. Выполните следующие действия, чтобы сделать альбом идей приватным на сайте Флатики:

* Войдите в свою учетную запись Флатики.
* Наведите курсор мыши на "Моя Флатика" в правом верхнем углу страницы.
* Нажмите на "Мои альбомы идей".
* Выберите альбом, который вы хотите сделать приватным.
* Нажмите на "Настройки" в правом верхнем углу страницы.
* Нажмите на кнопку рядом с "Скрыть альбом идей", чтобы сделать его приватной.
* Нажмите на кнопку "Сохранить", чтобы сохранить изменения.

Выполните следующие действия, чтобы сделать альбом идей приватным в приложении Флатики:

* Нажмите на значок сердца, чтобы перейти в раздел "Альбомы идей".
* Выберите альбом, который вы хотите сделать приватным.
* Перейдите к трем точкам в правом верхнем углу экрана.
* Выберите "Настройки" из выпадающего списка.
* Нажмите на кнопку рядом с надписью "Сделать Альбом идей скрытым / приватным".
* Нажмите на кнопку "Готово" в правом верхнем углу экрана, чтобы сохранить изменения.
* Вы увидите значок замка.

Обратите внимание, что вы можете выполнить те же действия, чтобы снять статус приватности с ваших альбомов идей, когда захотите.
