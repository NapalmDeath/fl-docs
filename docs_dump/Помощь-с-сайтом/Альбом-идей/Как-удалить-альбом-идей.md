# Как удалить альбом идей?

Если хотите удалить один из своих **альбомов идей**, сделайте так.

1. Войдите в свой профиль на **Флатике**.  
2. Наведите курсор на аватар в правом верхнем углу страницы и выберите **Альбомы идей**.  
3. Откройте альбом, который хотите удалить.  
4. Нажмите на иконку **урны** в правом верхнем углу.  
5. В появившемся окне подтвердите действие, выбрав **Удалить альбом**.

> После удаления альбом восстановить нельзя, поэтому убедитесь, что вы действительно хотите его удалить.
