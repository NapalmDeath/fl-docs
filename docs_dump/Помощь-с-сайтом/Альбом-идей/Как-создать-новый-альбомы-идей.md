# Как создать новый альбом идей?

Создать **альбом идей** на сайте просто!

1. Войдите в свой профиль на **Флатике**.  
2. Наведите курсор на аватар в правом верхнем углу страницы и выберите **Альбомы идей**.  
3. Нажмите **Создать альбом**.  
4. Придумайте название для нового альбома.  
5. При необходимости включите переключатель **Скрыть альбом идей**, чтобы сделать его приватным.  
6. Нажмите **Создать** — готово.

> Скрытые альбомы видите только вы. Их можно открыть для других пользователей в любое время.
