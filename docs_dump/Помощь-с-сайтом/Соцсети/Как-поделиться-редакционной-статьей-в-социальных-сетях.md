# Как поделиться статьей в социальных сетях?

> Статьями можно делиться через социальные сети.
{.is-info}

  
Чтобы поделиться редакционной статьей **на сайте**, просто нажмите на кнопку **Поделиться** над статьей и в появившемся окне  выберите способ, через который вы хотите поделиться статьей.
