# Добро пожаловать в Гайд по "Флатике"

**Знакомо?**

- "Как мне..."
- "Почему не работает..."
- "Где найти..."

**Отлично!** Вы в нужном месте. В нашем гайде по "Флатике" мы собрали решения для самых популярных сценариев и подготовили четкие инструкции.

Найдите вопрос → Изучите ответ → Примените решение

## Выберите свой путь к ответу

Мы предусмотрели несколько вариантов поиска — выбирайте самый удобный для вас:

**1. Через поисковую строку:**

- Используйте поисковую строку в верхней части страницы
- Вводите ключевые слова из вашего вопроса
  > _Важно: поиск чувствителен к регистру букв и к окончаниям слов. Используйте начало вопросов с заглавной буквы ("Как добавить..." , "Скачивание...") или ключевые слова без окончаний ("альбом", "счет")_
  > {.is-warning}

**2. Через боковое меню:**

- Слева доступны все основные разделы Гайда по "Флатике"
- Кликните на раздел, чтобы увидеть список его подразделов или конкретных страниц (отображаются под заголовком "Текущий каталог" на боковом меню)
- Выберите страницу для просмотра ответа на интересующий вопрос

**3. Через обзор основных разделов вопросов (ниже на этой странице):**

- Изучите список основных разделов с описанием к каждому
- Выберите подраздел, соответствующий вашей проблеме (для каждого подраздела представлено краткое описание)
- Изучите список вопросов в выбранном подразделе
- Перейдите к интересующему вопросу для получения подробного ответа

Если вам всё же не удаётся найти ответ на свой вопрос, [напишите в поддержку](mailto:help@flatica.ru){.is-primary} — поможем разобраться.
