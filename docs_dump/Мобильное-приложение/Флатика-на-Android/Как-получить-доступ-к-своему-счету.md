# Как получить доступ к своему счету?

Для входа в аккаунт Флатики на мобильном устройстве есть две опции:

* Нажмите "Продолжить с помощью электронной почты" и введите адрес электронной почты и пароль, которые вы использовали при регистрации аккаунта.
* Нажмите "Продолжить с помощью Google", если устройство помнит ваши учетные данные для входа.
