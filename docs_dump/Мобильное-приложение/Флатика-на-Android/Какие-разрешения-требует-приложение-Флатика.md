# Какие разрешения требует приложение Флатика?

Когда вы устанавливаете приложение Флатика через Google Play или App Store, вы сможете увидеть, какие разрешения требует Флатика. Вот для чего используются эти разрешения:

* Камера
  + Снимать фотографии и видео.
* Контакты
  + Доступ к контактам.
  + Находить учетные записи на устройстве.
* Местоположение
  + Доступ к точному местоположению.
  + Доступ к приблизительному местоположению (на основе сети).
* Микрофон
  + Запись звука.
* Телефон
  + Считывание статуса телефона и идентификационных данных.
* Хранилище
  + Изменение или удаление содержимого общего хранилища.
  + Чтение содержимого общего хранилища.
* Другое
  + Запустить сервисы.
  + Запуск при запуске.
  + Читать уведомления о значках.
  + Просматривать подключения.
  + Предотвращение перехода телефона в спящий режим.
  + Просмотр подключений Wi-Fi.
  + Получать данные из Интернета.
  + Чтение конфигурации службы Google.
  + Управлять вибрацией.
  + Иметь полный доступ к сети.
  + Воспроизводить Install Referrer API.
  + Изменять настройки звука.

Обратите внимание, что доступ к этим разрешениям можно отключить в Настройках. Обновления Флатика могут автоматически добавлять дополнительные возможности в каждой группе.
