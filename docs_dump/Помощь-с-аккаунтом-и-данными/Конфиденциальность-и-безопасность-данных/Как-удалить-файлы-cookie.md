# Как удалить файлы cookie?

Чтобы узнать, как удалить файлы cookie из браузера, перейдите по ссылкам ниже. Имейте в виду, что очистка всех файлов cookie в вашем браузере очистит для вас историю и cookie всех сайтов, которые вы посещали — не только «Флатики».

* [Google Chrome](https://support.google.com/chrome/answer/2392709)
* [Safari](https://support.apple.com/guide/safari/manage-cookies-sfri11471/mac)
* [Mozilla Firefox](https://support.mozilla.org/ru/kb/udalenie-kukov-i-dannyh-sajtov-v-firefox)
* [Microsoft Edge](https://support.microsoft.com/en-us/microsoft-edge/delete-cookies-in-microsoft-edge-63947406-40ac-c3b8-57b9-2a946a29ae09#:~:text=Delete%20all%20cookies,-In%20Edge%2C%20select&text=Select%20Settings%20%3E%20Privacy%2C%20search%2C,and%20then%20select%20Clear%20now.)
* [Opera](http://help.opera.com/Linux/10.50/ru/cookies.html)
