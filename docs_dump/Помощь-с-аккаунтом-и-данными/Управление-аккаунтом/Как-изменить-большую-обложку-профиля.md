# Как добавить большую фотографию обложки в свой профиль?

Хотите обновить оформление профиля? Добавьте красивую обложку — это просто


## Как загрузить фото обложки?

1. Откройте страницу своего профиля на **Флатике** и перейдите в **Настройки профиля**.  
2. В разделе **Основная информация** найдите блок **Оформление**.  
3. Нажмите **Изменить**.  
4. Загрузите новое фото с устройства или просто перетащите его в окно загрузки.  
5. При необходимости подвиньте или обрежьте кадр.  
6. Нажмите **Сохранить** — готово!


## Рекомендации по размеру фото

- Оптимальный размер обложки — **1200 × 312 px**.  
- Для экранов с высокой плотностью пикселей (Retina, 4K) используйте изображение **до 2192 × 624 px** — так оно будет выглядеть максимально четко.  

> Если обложка обрезается, попробуйте выбрать изображение без мелких деталей по краям.
