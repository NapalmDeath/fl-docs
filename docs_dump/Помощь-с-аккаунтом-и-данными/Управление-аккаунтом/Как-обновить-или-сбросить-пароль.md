# Как обновить или сбросить пароль

Если забыли пароль — ничего страшного. Вот что нужно сделать:

1. Перейдите на **страницу входа** на «Флатику».  
2. Нажмите **Забыли пароль?**  
3. Откроется страница **Восстановления пароля**.  
4. Введите email, указанный при регистрации.  
5. Перейдите по ссылке из письма, чтобы задать новый пароль.

> Готово! Пароль обновлён — теперь можете войти в аккаунт с новыми данными.
{.is-success}




## Требования к паролю

Ваш пароль должен содержать:

* От 8 до 40 символов.  
* Заглавные и строчные буквы.  
* Хотя бы одну цифру **или** специальный символ из списка:  
  `$ % & ( ) * + , - . / : ; < = > ? @ [ \ ] ^ _ { | } ~`

> Пробелы использовать нельзя.
> {.is-info}
